#Примечание(для проверяющих)
			
Изначально задумывалось использовать Express(легковесный) и twig(с ним работал раньше), но при реализации авторизации столкнулся с проблемой - без костылей и ухищерений невозможно передать данные из twig-шаблонов в Express для дальнейшей обработки. 

Поэтому моё желание сделать быстрее наоборот меня задержало, и для ускорения разработки был взят за основу проект из туториала:
https://www.smashingmagazine.com/2020/10/authentication-in-vue-js/

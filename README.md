# CitrusTestTask
Структура:
- Docs
- CheckListServer
- CheckListApplication

## Docs
Содержит документацию в виде drawio-схем: 
1. DDD-схемы:
	- Модель предметной области.drawio
	- Контексты DDD.drawio
	
2. Схемы из концепции C4Model
	- Контексты C4Model.drawio
	- Контейнеры.drawio
	- Диаграммы компонентов
		- Чек-лист сервер.drawio
		- Приложение Сборник чек-листов.drawio
		
Примечание
1. DDD-схемы описывают предметную область, то есть в общих чертах, что нужно сделать и как это примерно организовать, без подробностей реализации.
2. C4Model схемы ближе к коду и описывают устройство на более подробном уровне. Диаграмму классов решил не делать, так как все сущности и как их организовать описаны в "Docs/Контексты DDD.drawio".
